
# Girlzone AI

A Next.js application with AI features, payment processing, and user authentication.

## Quick Start

1. Clone the repository
```bash
git clone <your-repo-url>
cd girlzone
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env.local
```
Fill in your API keys in `.env.local`

4. Run development server
```bash
npm run dev
```

Visit http://localhost:3000

## Deploy to Vercel

1. Push your code to GitHub
2. Import project in Vercel
3. Add environment variables in Vercel dashboard
4. Deploy

## Required Environment Variables

See `.env.example` for the full list of required variables including:
- Stripe keys
- Database credentials
- API keys for AI services
- Cloudinary configuration

## Admin Access

Default admin setup key: Check your environment variables

## Support

For issues, please open a GitHub issue.
